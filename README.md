# Planmill Sum

Read a hour report exported from Planmill, add a total sum column and output the total sum of billable hours in the report.

## Compile

```
lein uberjar
```

## Usage

```
$ java -jar target/uberjar/planmill-sum.jar export.xml output.xlsx
```
